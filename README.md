# TaskZen — 極簡任務管理系統 ✦

具有禪意美學的任務管理工具，專注於簡單、高效、優雅的任務組織。

## 功能特色

### 📋 看板管理
- **三欄 Kanban**：待辦 → 進行中 → 已完成
- **拖放操作**：直覺拖曳卡片變更狀態與排序
- **即時篩選**：全部任務/緊急/今日到期/指派給我

### 🏷️ 任務分類
- 支援多種分類：開發、設計、研究、會議、個人
- **艾森豪矩陣四象限**：
  - 🔴 **Q1 重要且緊急** — 立即執行
  - 🔵 **Q2 重要不緊急** — 排程規劃
  - 🟠 **Q3 緊急不重要** — 委派他人
  - ⚪ **Q4 不重要不緊急** — 刪除或擱置
- 指派負責人、設定截止日期

### 📊 分析儀表板
- 總任務數、完成率、緊急任務、逾期項目統計
- 優先級分佈圓餅圖
- 分類分佈長條圖
- 近期任務表格檢視

### 🔄 即時同步
- 使用 BroadcastChannel API
- 多分頁同步更新
- 本地 localStorage 持久化

### 📱 跨平台支援（PWA）
- 離線存取
- 可安裝至主畫面（手機/桌面）
- 響應式設計（手機/平板/桌機）

## 技術亮點

| 項目 | 說明 |
|------|------|
| **極簡 UI** | 禪意美學設計系統：暖紙色調、留白節奏、細膩陰影 |
| **拖放功能** | SortableJS 實現流暢拖曳排序 |
| **即時同步** | BroadcastChannel 跨分頁即時更新 |
| **資料分析** | Chart.js 動態圖表儀表板 |
| **離線支援** | Service Worker 快取策略（網路優先） |
| **零建置** | 單頁 HTML，無需任何建置工具 |

## 快速開始

```bash
# 使用任何靜態檔案伺服器
cd task-zen
npx serve .
# 或
python3 -m http.server 8080
```

然後在瀏覽器開啟 `http://localhost:8080` 即可使用。

## 專案結構

```
task-zen/

├── manifest.json   # PWA Manifest
├── sw.js           # Service Worker
```

## 鍵盤快捷鍵

- `⌘/Ctrl + N`：新增任務
- `Esc`：關閉 Modal

## 設計理念

TaskZen 的設計核心來自日本禪學的「侘寂」（Wabi-sabi）美學：

- **簡素（Kanso）**：去除多餘裝飾，保留必要功能
- **留白（Ma）**：充足的負空間，呼吸感
- **寂靜（Shibui）**：內斂的色彩，低調的優雅
- **自然（Shizen）**：流暢的互動，有機的動畫

色彩靈感來自傳統和紙（Washi）、墨（Sumi）、土壁（Tsuchikabe），傳遞溫潤、安定的視覺感受。
